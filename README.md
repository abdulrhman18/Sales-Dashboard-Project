# Sales Performance Analysis Dashboard

## 📊 Project Overview
This interactive **Power BI Dashboard** provides a deep-dive analysis of global sales operations. It transforms raw sales data into actionable business intelligence, allowing stakeholders to monitor revenue trends, salesperson performance, and regional market health.

## 🚀 Key Metrics (KPIs)
* **Total Revenue:** $30.09M
* **Total Quantity Sold:** 86K units
* **Average Selling Price:** $350
* **Tax & Freight:** $2.93M (Tax) | $915.97K (Freight)
* **Customer Base:** 294 Unique Customers

## 🔍 Key Insights
The dashboard is structured into three primary analytical views:

### 1. Executive Summary
* **Revenue Growth:** Performance tracking from 2011–2014, highlighting a peak growth period in late 2013.
* **Product Dominance:** **Bikes** account for over **80% ($24.23M)** of total revenue.
* **Geographic Leaders:** **Canada** leads all territories with $10.6M in sales, followed by the **Northwest** region ($6.2M).

### 2. Sales & Product Intelligence
* **High-Value Staff:** Identification of top-tier sales personnel, led by SalesPerson ID 289 ($8.5M).
* **Sub-Category Performance:** Road Bikes ($9.1M) and Mountain Bikes ($8.5M) are identified as the core revenue drivers.

### 3. Operational Efficiency
* **Territory Distribution:** Analysis of order volume vs. salesperson headcount per region to identify under-served markets.
* **Volume Analysis:** Breakdown of 86K units sold across Accessories, Clothing, Components, and Bikes.

## 🛠️ Tech Stack
* **Power BI Desktop:** For data modeling and visualization.
* **DAX (Data Analysis Expressions):** Used for advanced calculations (Total Revenue, YoY Growth, etc.).
* **Power Query:** For ETL (Extract, Transform, Load) processes and data cleaning.

## 📂 How to View
1. Download the `.pbix` file from this repository.
2. Open the file using **Power BI Desktop**.
3. Use the navigation arrows in the header to switch between the **Summary**, **Insights**, and **Operational** pages.
